# Memorama Justo

## Tech/framework used
* React
* Firebase
* Styled Components

## Installation
Clone repository
```git clone ...```

Install dependencies
```npm i```

Run project in development mode
```npm run start```

You can see the project in:
```http://localhost:3000```

Run project for production
```npm run build```
